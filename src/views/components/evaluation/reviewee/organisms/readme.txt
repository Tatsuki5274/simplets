moleculesを集合させたコンポーネント
画面ごとに階層を分けて管理する。

[許可]
・mutationの実行
・任意の型を引数に要求するコンポーネントの作成
・親ステートへのset
・条件分岐による振る舞いの分岐
・molecules,atomsの呼び出し、及び固有のスタイリング
・コンポーネント依存の型定義のexport及び定義

[禁止]
・自身の大きさに関するスタイルを持つこと
・queryの実行

[ディレクトリ]
・ルートへのコンポーネント追加は禁止
・機能や目的別に階層を作成可能
・階層数に制限なし
・./report/create, ./report/list, ./common

[props]
・任意の型を持つ
